# claude-code-prompts

## 核心原理
就像给AI写作的提词器，这些提示模板让AI更好理解并执行编程任务，提升效率和准确度。

## 主要用例
- 文科生学习AI编程辅助语言
- AI新手优化与AI代码助手的交互
- 产品经理设计智能提示系统

## 为什么会火
优质提示语是提升AI工具体验关键，独立作者贡献丰富模板满足开发者提升工作效率需求。

## 对 AI 产品经理的价值
理解提示工程的重要性，提升AI产品的用户交互设计能力。

## 产品机会
