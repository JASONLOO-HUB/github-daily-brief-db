# claude-code-haha

## 核心原理
像一个本地版的代码运行环境，方便你随时执行代码。

## 主要用例
- 文科生本地测试代码学习
- 辅助转行者搭建个人代码环境
- 用来练习代码运行与调试

## 为什么会火
本地可运行版本满足安全和便捷需求。

## 对 AI 产品经理的价值
偏转行复盘，分享环境搭建和本地开发经验，适合进阶文科生学习。
