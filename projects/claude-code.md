# claude-code

## 核心原理
基于泄露的NPM映射文件，解析和拆解Claude Code源码，方便理解和分析。

## 主要用例
- 对Claude Code源码进行逆向分析
- 学习项目架构和功能设计
- 帮助社区进行安全和质量审查
