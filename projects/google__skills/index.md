# google/skills

## 稳定定位

Google 官方维护的 Skills 仓库，把 Google 产品、工具和技术相关知识封装成 Agent 可读的任务能力。

## 原理解释

大型组织内部有大量“怎么用这个产品、怎么排错、怎么产出结果”的隐性知识。Skills 的价值是把这些知识变成机器也能遵循的说明，让 Agent 在面对具体任务时少走弯路。

## 典型用例

- 开发者让 Agent 更熟悉 Google 生态工具。
- 企业把内部 SOP 和平台使用方式做成私有 skills。
- PM 研究官方生态如何把文档升级为可执行上下文。

## 对 AI 产品经理的价值

这是官方生态入局信号：未来文档、模板、最佳实践可能不只给人读，也会直接给 Agent 调用。

## 对小红书账号的价值

适合讲“为什么大厂开始给 AI 写说明书”，把文档产品化、知识结构化讲给非工程读者。

## 产品化机会

做企业内部“Agent 可读知识库”：把流程文档、产品手册、客服 SOP 转成可执行 Skills。

## 风险与限制

官方 skills 更偏特定生态。对国内用户要注意可访问性、账号体系和本地工具替代方案。
