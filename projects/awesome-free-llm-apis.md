# mnfst/awesome-free-llm-apis

## 核心原理

一个持续维护的免费 LLM API 清单，汇总了 OpenAI、Anthropic、Google Gemini、国内各大模型等提供的永久免费额度或免费套餐，并注明了每个 API 的限制条件（速率限制、Token 上限等）。

## 主要用例

- 独立开发者零成本构建 AI 应用原型
- 学生和研究者进行 AI 实验
- 创业团队在早期阶段降低 API 成本

## 为什么会火

AI API 成本是所有想做 AI 产品的人面临的第一个门槛。这个列表直接解决了"我想做 AI 产品但没钱付 API 费"的痛点。

## 对 AI 产品经理的价值

了解免费 API 生态，是快速验证产品想法的基础。你可以利用这些免费 API，在不花一分钱的情况下构建产品 Demo，再去验证用户需求。
