# datawhalechina/easy-vibe

## 稳定定位

easy-vibe 是 Datawhale 中文社区的 vibe coding 学习项目，面向想用 AI 辅助完成编程项目的学习者。它的价值在于把“不会写代码也能做项目”的过程课程化，而不是只给工具清单。

## 原理解释

vibe coding 的关键不是完全不懂代码，而是把需求表达、AI 生成、运行验证、错误修复和交付验收串成循环。对非工程背景学习者来说，重点是学会描述目标、拆步骤、检查结果，而不是一开始背语法细节。

## 典型用例

- 文科生用 AI 做个人网页、数据小工具或自动化脚本。
- AI 教育账号把每节课设计成“一个可交付项目”。
- 产品经理用它理解从需求到 demo 的最短路径。

## 对 AI 产品经理的价值

它展示了 AI 学习产品的转向：从知识点课程转向项目制、任务制、验收制。PM 可以借鉴其教学路径，把“学工具”设计成“完成一个具体成果”。

## 对小红书账号的价值

非常适合转行人群选题：普通人学 AI 编程，不是先成为程序员，而是先学会让 AI 帮自己做一个小产品。

## 产品化机会

MVP 可以做“AI 项目练习营”：每周一个小项目，提供任务卡、提示词、运行环境、验收表和踩坑直播。差异化是把 GitHub 项目转成中文非工程学习路径。

## 风险与限制

vibe coding 容易让新手忽略代码质量、安全和维护。课程必须加入验收、调试、版本管理和隐私边界，否则容易停留在玩具 demo。
