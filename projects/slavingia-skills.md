# slavingia/skills

## 核心原理

由 Gumroad 创始人 Sahil Lavingia 发布的 Claude Code 技能集合，基于其著作《极简创业家（The Minimalist Entrepreneur）》的商业理念。这些技能（Skills）是 Claude Code 的扩展模块，可以让 AI 按照特定的商业框架（如精益创业、最小可行产品）来辅助创业决策和执行。

## 主要用例

- 独立开发者（Indie Hacker）使用 AI 快速验证商业想法
- 产品经理使用 AI 进行竞品分析和市场调研
- 创业者使用 AI 生成商业计划书和产品路线图

## 为什么会火

Sahil Lavingia 本人的影响力（Gumroad 创始人，独立开发者圈的标志性人物）+ Claude Code 生态的爆发式增长。这个项目代表了一个信号：**顶级创业者开始将自己的方法论"技能化"，注入 AI 工具**。

## 对 AI 产品经理的价值

"技能（Skill）"正在成为 AI 时代的新型知识产品。传统的知识变现是书籍、课程；AI 时代的知识变现是可以被 AI 直接调用的"技能包"。这对你来说是一个极其重要的产品机会。
