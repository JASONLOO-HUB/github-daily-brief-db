# api-relay-audit

## 核心原理

该工具专注检测第三方AI API代理服务中的安全隐患，如提示注入、信息泄露和指令覆盖，保障AI服务调用的安全与可信。

## 主要用例

- 审计AI API代理服务的安全风险
- 防止恶意提示注入攻击
- 保障AI产品数据隐私和准确性

## 为什么会火

随着AI服务广泛接入，安全问题日益突出，该工具针对性解决实际痛点，满足行业对安全合规的刚需。

## 对 AI 产品经理的价值

提醒产品经理安全是AI产品的重要环节，尤其涉及第三方API时需加强审计和风险控制能力。
