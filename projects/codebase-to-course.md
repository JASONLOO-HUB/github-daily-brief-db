# codebase-to-course

## 核心原理

这是一个 Claude Code 技能（Skill），其本质是一个 AI 驱动的"代码理解 → 课程生成"管道。它接收任意 GitHub 代码库作为输入，通过 LLM 自动分析代码结构、提取核心概念，最终输出一个精美的单页 HTML 交互式课程，包含章节导航、代码高亮和概念解释。专为非技术背景的"氛围程序员（Vibe Coders）"设计。

## 主要用例

- 非技术背景的产品经理快速理解一个开源项目的技术架构
- 开发者为自己的项目自动生成文档和教程
- 教育机构将开源项目转化为课程材料
- 文科生学习 AI 工具时，用于拆解复杂项目

## 为什么会火

切中了"Vibe Coding（氛围编程）"时代的核心需求：大量非技术人员开始使用 AI 写代码，但他们缺乏理解代码的能力。这个工具提供了一个"翻译层"，让代码变得可读、可学。

## 对 AI 产品经理的价值

揭示了一个重要的产品设计范式：**技术内容的"可读性层（Readability Layer）"**。未来会有越来越多的工具，专门负责把技术内容（代码、论文、API 文档）翻译成不同受众能理解的形式。这是一个巨大的蓝海市场。
