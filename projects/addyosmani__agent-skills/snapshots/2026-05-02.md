# 2026-05-02 Snapshot：addyosmani/agent-skills

## 为什么今天值得关注

**事实**：仓库集中收录面向 Agent 的工程 Skill，并提供站点和 release 入口。第三方搜索结果显示其 fork 数已到千级。

**推断**：Agent Skills 正在变成新的分发形态：用户未来可能不是买一个 AI 工具，而是给已有 Agent 安装“会做某类任务的能力包”。

## 当天变化与信号

- new_hot：工程 Skill 包被广泛转发，说明开发者开始关注“如何让 Agent 稳定执行流程”。
- PM 信号：Skill 市场需要解决发现、安装、版本、权限、安全和评价机制。
- 内容信号：这比“提示词合集”更高级，适合做转行教育。

## 典型用例

- 代码审查 Skill：让 Agent 按固定 checklist 找 bug。
- 性能 Skill：让 Agent 先测量，再优化，再复测。
- 发布 Skill：让 Agent 按 release note、测试、tag、回滚计划执行。

## 产品化机会

MVP：中文 Skill 目录站，按岗位分类：产品、运营、内容、求职、数据分析、开发。每个 Skill 给安装方式、输入样例、输出样例和风险提醒。

## 风险与限制

Skill 不是独立应用，需要依赖 Codex、Claude Code 或其他 Agent 执行。不同宿主支持的文件结构和工具权限可能不一致。
