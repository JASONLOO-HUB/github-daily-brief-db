# HKUDS/OpenSpace

## 核心原理

OpenSpace 是香港大学数据科学实验室（HKUDS）出品的多 Agent 协作框架。其核心特点是"低成本、自进化"——Agent 可以通过与环境交互，自动学习并优化自身的行为策略，而无需人工干预。框架支持多个 Agent 协同工作，共享知识库，实现复杂任务的分布式执行。

## 主要用例

- 构建能够自主学习的 AI 研究助手
- 企业级多 Agent 工作流自动化
- 学术研究中的 AI 实验平台

## 为什么会火

学术机构背书（港大）+ 解决了 Agent 框架的核心痛点（成本高、不能自进化）。

## 对 AI 产品经理的价值

**自进化 Agent（Self-Evolving Agent）** 是 Agent 技术的下一个重要里程碑。当前大多数 Agent 框架都是静态的（规则固定），而 OpenSpace 展示了 Agent 如何通过经验积累来改进自身。这对于构建长期运行的 AI 产品（如个人 AI 助理、企业知识库 Agent）具有重要意义。
