# AI Memory

## 关联项目

- [MemPalace/mempalace](../projects/mempalace__mempalace/index.md)

## 产品观察

AI 记忆的关键不只是“存聊天记录”，而是让 agent 在新任务里能找回原始上下文、人物关系、项目历史和时间线。长期看，记忆层会决定 AI 工具是否能从“每次重新解释”变成“越用越懂你”。

## 小红书切入

- “为什么 AI 每次都像第一次认识你？因为你缺一个 AI 记忆系统。”
- “用知识宫殿理解 AI 长期记忆：人、项目、主题、原文证据怎么组织。”
