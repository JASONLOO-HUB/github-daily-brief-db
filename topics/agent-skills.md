## 产品观察

Agent Skills 把“会做某件事”拆成可安装、可复用、可审计的工作规程。它通常包含 `SKILL.md`、示例、脚本、资源和验收标准，让同一个 Agent 在不同项目里复用同一套做事方式。

**事实**：2026-05-02 入选项目中，open-design、agent-skills 和 guizang-ppt-skill 都把 Skill 作为核心组织方式。
**推断**：普通用户未来不会直接学习复杂模型参数，而会购买、安装或订阅“帮我做某类任务的 Skill 包”。
