## 本周最推荐产品机会

### 机会一：GitHub 项目中文解读生成器

**核心逻辑：** 利用 `codebase-to-course` 的思路，做一个网页工具，输入任意 GitHub 项目链接，输出一篇面向文科生的中文解读文章（可直接发布到小红书/公众号）。

**为什么现在做：** 技术门槛低（调用现有 LLM API）、用户需求明确（你的粉丝就是目标用户）、可以边做边发内容。

**执行路径：**
1. 用 Cursor/Claude Code 搭建一个简单的网页（1-2天）
2. 调用免费 LLM API（参考 `awesome-free-llm-apis`）
3. 将搭建过程发布为小红书连载内容
4. 产品上线后，作为粉丝福利推广

---

### 机会二：中文版 Claude Code Skills 集合

**核心逻辑：** 将 `slavingia/skills`、`wewrite`、`codebase-to-course` 等英文 Skills 翻译并本地化，加入微信、小红书、抖音等中国场景的 Skills。

**为什么现在做：** 这是一个极低竞争的蓝海，中文互联网上几乎没有人在做。

**执行路径：**
1. 在 GitHub 创建 `chinese-claude-skills` 仓库
2. 翻译并本地化现有 Skills
3. 每发布一个 Skill，就在小红书发一篇教程
4. 逐步建立"中文 Claude Code 技能专家"的人设

## 产品策略：把做产品变成内容

> 核心理念：Build in Public（公开构建）

每一个产品的开发过程，都是一系列小红书内容的素材：
- 《我为什么要做这个产品》（选题/立项）
- 《我用 AI 写了第一行代码》（开发过程）
- 《第一个用户来了！》（上线）
- 《我的 AI 产品第一周：10个用户，3个付费》（复盘）

这种"养成系"内容极具粘性，能吸引大量想要转行 AI 的粉丝，同时产品本身也在真实推进。
